WinCtrl is very easy to use. Here is a list of all its current uses.

 

 

Windows1.AboutWinCtrl     -     This will display the WinCtrl About Dialog

 

Windows1.GetCalculator – This opens the Calculator program

 

Windows1.GetCtrlPanel – Opens Control Panel

 

Windows1.GetFontDit – Opens the font folder

 

Windows1.GetNotepad – Opens Notepad

 

Windows1.GetProgFiles – Opens Program Files folder

 

Windows1.GetStartMenu – Opens the Start Menu folder

 

Windows1.GetSysDir – Opens the System folder

 

Windows1.HowLongSession – Displays the length of your current Windows session

 

Windows1.LogOffWin – Logs Off

 

Windows1.OpenBrowse – Open a file you browse for

 

Windows1.OpenBrowser “[Your URL]” – Opens any site in the default browser

 

Windows1.OpenCD – Opens the CD-ROM drive

 

Windows1.OpenExeCopy – Opens another copy of the program

 

Windows1.OpenPath “[Your Path]” – Opens any path

 

Windows1.RestartWin – Restarts Windows

 

Windows1.ShowAboutBox “[Your Text]”,”[Your Product]” – Shows the custom About dialog

 

Windows1.ShutDownWin – Shuts Down Windows

 

 

Enjoy using this powerful, time-saving control!
